## 💡 마케팅 전략

### 1. 이메일 마케팅 시나리오

**시나리오 1: 구독 시**
```
즉시: Welcome + 50% 할인 쿠폰
→ 전환율 15%
```

**시나리오 2: 검사 중단**
```
5분 후: 리마인더 + 10% 추가 할인
→ 전환율 20%
```

**시나리오 3: 무료 검사 완료**
```
즉시: 프리미엄 업그레이드 제안 (50% 할인)
3일 후: 성공 사례
7일 후: 마지막 기회 (70% 할인)
→ 전환율 15%
```

**시나리오 4: 주간 뉴스레터**
```
매주 화요일: 심리학 인사이트 + 30% 쿠폰
→ 전환율 5%
```

### 2. 카카오톡 챗봇 시나리오

**시나리오 1: 첫 대화**
```
사용자: "안녕"
챗봇: Welcome + Quick Reply (검사 추천/가격/결과)
```

**시나리오 2: 검사 추천**
```
챗봇: 인기 검사 TOP 3 + 간단 설명
→ Quick Reply (MBTI/에니어그램/EQ)
```

**시나리오 3: 가격 문의**
```
챗봇: 가격표 + 50% 할인 안내
→ Quick Reply (지금 구매/무료 검사)
```

### 3. AI 상담 시나리오

**주제별 응답:**
- 💼 진로: MBTI + 홀랜드 검사 추천
- 💑 연애: 애착유형 + MBTI 궁합
- 👥 대인관계: DISC + EQ 검사
- 😰 스트레스: 정신건강 자가진단
- 🎯 자기계발: 성격 유형별 팁
