## 🎯 프로젝트 개요

**MindScan V3.0**은 **독특한 그린 + 골드 색상**과 **직관적인 검색 시스템**을 결합한 **프리미엄 심리검사 플랫폼**입니다.

> **운영**: 큐브박스 (CubeBox)  
> **서비스**: MindScan

### 💎 핵심 특징

- ✅ 12가지 전문 심리검사 (MBTI, 에니어그램, EQ, IQ, SQ 등)
- ✅ 실시간 검사 로직 (MBTI 완성)
- ✅ 스마트 검색 시스템 (90% 정확도)
- ✅ 관리자 대시보드
- ✅ 고객 마이페이지
- ✅ 그린 + 골드 프리미엄 디자인
- ✅ 완벽한 반응형 디자인

---

## 🚀 주요 기능

### 1. 검사 시스템
- MBTI 성격유형 검사 (12문항)
- 16가지 유형별 상세 결과
- LocalStorage 자동 저장
- 결과 공유 기능

### 2. 관리자 페이지
- 실시간 통계 대시보드
- 사용자 관리
- 검사 통계
- Chart.js 데이터 시각화

### 3. 고객 페이지
- 마이페이지
- 검사 이력 관리
- 프로필 설정
- 프리미엄 리포트
