### 🔬 심리검사 시스템

  - MBTI 성격유형검사 (16유형)
  - 에니어그램 (9가지 성격 유형)
  - Big Five 성격 검사

  - 애착유형검사

  - MBTI 12문항, 16가지 유형 분석
  - 즉시 결과 확인
  - LocalStorage 자동 저장
  - 결과 공유 기능
