🎯 ISFJ란?

ISFJ (Introverted, Sensing, Feeling, Judging)

  • I (내향): 조용하고 신중하며, 깊은 관계를 중시
  • S (감각): 현실적이고 실용적, 디테일 중시
  • F (감정): 따뜻하고 공감 능력이 뛰어남
  • J (판단): 계획적이고 체계적, 안정 추구

🧠 인지 기능 스택

  1. 주 기능 - Si (내향 감각): 과거 경험을 토대로 현재를 이해
  2. 부 기능 - Fe (외향 감정): 다른 사람의 감정에 민감하게 반응
  3. 3차 기능 - Ti (내향 사고): 논리적 분석 (덜 발달)
  4. 열등 기능 - Ne (외향 직관): 새로운 가능성 탐색 (약점)
